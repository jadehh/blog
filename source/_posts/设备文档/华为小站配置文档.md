## 3. 时钟同步

### 3.1 时区设置
> 进入管理平台，进入管理界面,点击时钟同步，设置时区,并点击同步按钮即可。
![image-20260717100707301](https://jadehh.github.io/blog/article/20240108164645/image-20260717100707301.png)

### 3.2 命令行同步时钟
```bash
systemctl stop ntpd

systemctl start ntpd
```

## 4. 分配磁盘
### 4.1 创建分区
![33b042cd377ab8d4fdb4d8520af72dc8.png](https://jadehh.github.io/blog/article/20240108164645/be43934928ca4e74b510af5d4e132d7c.png)  
填写磁盘大小  
![9b477c32b0eb1f7c81aba6756ca4cde6.png](https://jadehh.github.io/blog/article/20240108164645/11befc70314749529e284a9e4a64153b.png)

> 分区分为2个区，一个用来挂载数据，一个用来挂载Docker，磁盘如果大于4TB，每个分区大小为2048GB。
> 如果只有900多GB，则只需要分一个分区即可，大小为900多GB即可。

分区结果如下图：
![4094d658776137b73cb589b355f44cf.png](https://jadehh.github.io/blog/article/20240108164645/4094d658776137b73cb589b355f44cf.png)

创建分区如果遇到下面错误

![image-20260630092233606](https://jadehh.github.io/blog/article/20240108164645/image-20260630092233606.png)

进入deveop模式，输入下面的命令

```bash
/opt/middleware/MindXOM/bin/om_ability_policy.sh allow --disk_ops
systemctl restart ibma-edge-start.service
```

> 打开磁盘操作的能力项

### 4.2 磁盘挂载
> 进入develop模式挂载磁
![f7ba026d107667b62c750544449499b1.png](https://jadehh.github.io/blog/article/20240108164645/ad349b8e9aaf4ed293e864adc0de3c58.png)
```bash
mkdir /home/samples && mkdir /root/Docker && mkdir /home/samples/sda2
echo "/dev/hdisk0p1  /home/samples/sda2 ext4  defaults  0             0" >> /etc/fstab
echo "/dev/hdisk0p2  /root/Docker       ext4  defaults  0             0" >> /etc/fstab
sudo mount -a 
#赋予权限
chmod 777 /home/samples/sda2
```
注意，上面这种方式是有针对机械硬盘至少有4TB以上的，如果是固态硬盘只有900多GB，只需要分一个分区即可

```bash
mkdir /home/samples && mkdir /home/samples/sda2
echo "/dev/mdisk0p1  /home/samples/sda2 ext4  defaults  0             0" >> /etc/fstab
sudo mount -a 
chmod 777 /home/samples/sda2
```

>  这里的hdisk0p*和mdisk* 对应的就是网页中分区的磁盘，固定硬盘不在需要修改Docker路径

## 6. 修改IP地址

### 6.1 强制修改ip地址
```bash
vi /etc/sysconfig/network-scripts/ifcfg-eth1
```
```text
BOOTPROTO=""
DEVICE="eth1"
IPADDR="192.168.40.235"
NETMASK="255.255.255.0"
GATEWAY="192.168.40.254"
STARTMODE="auto"
ARPCHECK="no"
```
重启网络
```bash
/etc/init.d/network restart
```
> 强制修改ip后无法通过修改的ip登录管理平台，如需登录，切换至网口1，使用192.168.2.111进行登录。
### 6.2 修改DNS
```bash
rm -r /etc/resolv.conf  && echo "nameserver 218.2.135.1" > /etc/resolv.conf
```
> 看实际情况修改DNS



## 7. 安装加密狗驱动

需要使用[FileZilla](https://my.feishu.cn/file/boxcnQH9Hq4sWUORtjGk9e8jHcU) 将驱动文件上传到/tmp目录(只有tmp目录有权限,并且需要开启stfp enable)

通过SSH进入develop模式 cd 到 /tmp目录操作

### 7.1 驱动安装

```bash
rpm -ivh aksusbd-8.51-1.aarch64.rpm && rm aksusbd-8.51-1.aarch64.rpm
```

### 7.2 加密狗配置远程访问

```bash
 mv hasplm.ini /etc/hasplm/ \
 && service hasplmd.service restart
```

> 账号为：admin，密码为：Samples@123

## 8. 部署

需要将Docker文件上传到/tmp目录

```bash
docker load -i algorithm-cann6.tar
```


```bash
docker run \
  --name algorithm-cann6 \
  -itd \
  --net host \
  --privileged \
  --ulimit nofile=65536:65536 \
  --ulimit nproc=65536:65536 \
  --tmpfs /run --tmpfs /run/lock \
  -v /sys/fs/cgroup:/sys/fs/cgroup:ro \
  --device=/dev/davinci0 \
  --device=/dev/davinci_manager \
  --device=/dev/hisi_hdc \
  --device=/dev/devmm_svm \
  -v /usr/local/bin/npu-smi:/usr/local/bin/npu-smi \
  -v /var/hasplm:/var/hasplm -v /etc/hasplm:/etc/hasplm \
  -v /home/data/miniD/driver:/home/data/miniD/driver \
  -v /etc/localtime:/etc/localtime:ro \
  -v /run/board_cfg.ini:/run/board_cfg.ini \
  -v /home/samples/sda2:/home/samples/sda2 \
  -v /usr/sbin/ntpdate:/usr/sbin/ntpdate \
  -v /usr/lib64/libopts.so.25:/usr/lib/libopts.so.25 \
  --restart=always \
  jadehh/algorithm:1.0.0-cann6 /sbin/init
```

> 注意/home/samples/sda2 是磁盘挂载的目录
> 接下来使用算法配置界面进行部署，SSH端口号为：2222
